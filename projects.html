Hobb

Hobb es una red social tanto para apasionados del buen comer como para restaurantes a los que les permite obtener un análisis rápido sobre sus platillos, sus instalaciones y tipos de clientes para una mejor planificación del negocio.

En este proyecto participe como Software Engineer

Desarrollada en Java 11, Spring Boot, JHipster, Angular y Android
